订单状态
订单状态解释
“待付款”
我们尚未收到该订单的款项，请尽快付款。此时您可以选择在“我购买的商品”中点击“付款”按钮完成该订单或点击取消按钮“取消该订单”
“已付款”
该订单已成功付款，但是订单尚在处理中，商品未发货
“待收货”
该订单已在途中，买家等待确认收货
“已完成”
订单完成，买家已经确认收货的订单
“已取消”
下单后1小时内买家未付款成功，系统自动取消的订单
买家付款后主动申请取消的订单
异常订单，如用户付款后系统出现问题导致的无效订单
